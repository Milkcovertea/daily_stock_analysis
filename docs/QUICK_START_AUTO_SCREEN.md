# 🚀 自动股票筛选 - 5 分钟快速配置清单

> 按照这个清单，5 分钟内完成自动股票筛选功能配置并开始使用。

## ✅ 配置前准备

- [ ] 已 Fork 本仓库
- [ ] 已配置 AI 模型（Anspire/Gemini/OpenAI 等）
- [ ] 已配置至少一个通知渠道（企业微信/飞书/邮件等）
- [ ] 了解你的实盘持仓股票代码（可选，用于基础池）

---

## 📝 步骤 1：配置基础股票池（2 分钟）

### 1.1 进入配置页面

1. 打开你的 GitHub 仓库
2. 点击 **Settings** 标签
3. 左侧菜单选择 **Secrets and variables** → **Actions**
4. 点击 **Repository secrets** 标签页

### 1.2 添加基础股票池 Secret

点击 **New repository secret** 按钮：

```
Name:  BASE_STOCK_LIST
Value: 600519,000001,300750  # 替换为你的实盘持仓股票代码
```

**说明**：
- 多个股票代码用逗号分隔
- 如果不配置此项，将完全依赖自动筛选
- 建议配置 2-5 只你长期持有的股票

点击 **Add secret** 保存。

---

## 📝 步骤 2：启用自动筛选（1 分钟）

### 2.1 切换到 Variables 标签页

在同一个页面，点击 **Repository variables** 标签页。

### 2.2 添加自动筛选开关

点击 **New repository variable** 按钮：

```
Name:  AUTO_SCREEN_ENABLED
Value: true
```

点击 **Add variable** 保存。

---

## 📝 步骤 3：调整筛选参数（可选，1 分钟）

继续在 **Repository variables** 中添加：

### 3.1 调整最大收盘价（可选）

```
Name:  SCREENER_MAX_PRICE
Value: 50  # 默认 50 元，可调整为 30-100 之间
```

### 3.2 调整筛选数量（可选）

```
Name:  SCREENER_TOP_N
Value: 5  # 默认 5 只，可调整为 3-10 之间
```

**参数说明**：
- `SCREENER_MAX_PRICE`：只筛选收盘价低于此价格的股票
- `SCREENER_TOP_N`：按成交额排序，返回前 N 只股票

---

### 模式 1：完全自动筛选（无基础池）

```yaml

# Variables

SCREENER_MAX_PRICE: "50"
SCREENER_TOP_N: "5"
```

**效果**：每天筛选 5 只新股票


**效果**：5 只自动筛选 + 基础池股票（去重）

## 🔧 常见问题快速解决

### Q1: 筛选器报错 "No module named 'akshare'"

**解决**：依赖已在 `requirements.txt` 中，GitHub Actions 会自动安装。如果是本地测试，运行：

```bash
pip install -r requirements.txt
```

---

### Q2: 筛选结果为空

**可能原因**：
- 今天是周末或节假日（市场休市）
- 价格条件太严格（如 `SCREENER_MAX_PRICE` 太低）

**解决**：
- 等待下一个交易日
- 放宽价格条件（如提高到 60 元）

---

### Q3: 基础池股票未出现在最终列表

**原因**：基础池股票与自动筛选结果重复

**说明**：这是正常行为。重复的股票会使用自动筛选的详细信息（包含收盘价、成交额等），但代码不会重复。

---

### Q4: 如何查看筛选器生成的文件？

筛选器会生成两个文件（非 dry-run 模式）：

1. **JSON 报告**：`config/stocks.json`
2. **纯文本列表**：`config/stocks.txt`

在 GitHub Actions 中，这些文件会包含在上传的 artifacts 中：

1. 进入 Actions → 点击运行的任务
2. 页面底部找到 **Artifacts**
3. 下载 `analysis-reports-xxx`
4. 解压后查看 `config/` 目录

---

### Q5: 如何调整筛选策略？

**简单调整**：通过 Variables 调整参数

```yaml
# 放宽价格限制
SCREENER_MAX_PRICE: "80"

# 多筛选一些股票
SCREENER_TOP_N: "10"
```

**深度定制**：修改 `scripts/stock_screener.py` 源码
